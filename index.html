HTML Various Tags

Tag Name Description
Common HTML Tags
<h1> Heading 1 Headings and sub-headings, <h1> - <h6> tags, are used to provide titles for sections of content. 
<a> Anchor Anchor, <a> tags, are used to link to internal pages, external pages or content on the same page.
<p> <span> <div> p, span, div Text and block tags, are used to specify different sections of content on a page. 
<em> <strong> Emphasize text Em and Strong tags, are used to Emphasize text. 
<br> Line Break Adding <br> will add a line break in the content. 
<ol> <ul> <li> Ordered and Unordered lists Ordered list <ol> are numbered and unordered lists <ul> are bulleted. List items are enclosed in <li> tags.  
<body> Body Any visible content should be placed within the <body> tags.  
<img> <video> Images and videos Images and videos can be added by linking to an existing source. <img> is a self closing tag while <video> is not.  
<head> Head - Info about the webpage The <head> tag contains information about the web page such as the tab title and links to CSS and JS files. 
<Title> Title of Webpage The <title> tag, always within the <head> tag - gives the website and tab its title.  
HTML Table Tags
<table> Table The wrapper element for all HTML tables.  
<thead> Table Head The set of rows defining the column headers in a table.  
<tbody> Table Body The set of rows containing actual table data.  
<tr> <td> Table Row and Table Data Table row and Table Data containers <td> is always nested inside of <tr> tags.  
<tfoot> Table Footer The set of rows defining the footer in a table.  
HTML Form Tags
<form> Form A form allows users to input information and send it.  
<input> Input To add fields for users to input information, use the <input> tag and set the type attribute to a field of your choosing.  
<select> Select A <select> element paired with a <option> element renders a dropdown list selection.  
<datalist> Datalist A <datalist> element is populated with <option> elements and works with an <input> to search through choices.  
<textarea> Textarea A <textarea> element is a text input field that has a customizable area.  
HTML Semantic Tags
<header>, <nav>, <main>, <footer> Semantic tags Semantic tags creat the basic structure of a webpage.
<section> Section Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article Holds content that makes sense on its own such as articles, blogs, comments, etc.
<aside> Aside Contains information that is related to the main content, but not required in order to understand the dominant information.
<figure> Figure Encapsulates all types of media like pictures, graphs, and tables.
<figcaption> Figure Caption Used to describe media in <figure>.
<video>, <embed>, <audio> Media files Tags used for different types of media files on a page.

HTML Attributes

Attribute Name Description
Common HTML Attributes
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
text type=text Setting type to "text" creates a single row field for text input.
password type=password Setting type to "password" creates a single row field that censors text input.
number type=number Setting type to "number" creates a single row field for number input.
range type=range Setting type to "range" creates a slider to select from a range of numbers.
checkbox type=checkbox Setting type to "checkbox" creates a single checkbox that can be paired with other checkboxes.
radio type=radio Setting type to "radio" creates a radio button that can be paired with other radio buttons.
submit type=submit Setting type to "submit" creates a submit button.
required Required Adding te required attribute to an input related element will validate that the input field has information in it.
min & max Minimum & Maximum Assigning a value to the min or max attribute of a number input element will validate an acceptable minumum or maximum value.
minlength & maxlength Minimum Length & Maximum Length Assigning a value to the minlength or maxlength attribute of a text input element will validate an acceptable minumum or maximum number of characters.
pattern Pattern Assigning a regex to pattern matches the input to the provided regex.

CSS Selectors

Selector Name Description
Common CSS Selectors
Type Type - often an HTML tag An HTML tag or element such as <h1> or <p> can be chained together, if unrelated need to seperate with a comma.
.Class class= inside HTML tag Class selectors must be prepended with a "." Class selectors are resusable.
#ID id= inside HTML tag ID selectors must be prepended with a "#" ID selectors can only be used once.
Attribute other elements listed within the HTML tags. Can select nested elements by seperating selectors with a space.
* Universal Selector All elements can be selected by using the universal selector.

CSS Properties

CSS Property Name Description & Common Selections
Common CSS Properties
border Can set pixel width, solid/dotted/dashed/ and color - no spaces
color Sets an element's color - for text - the text will be that color
bacground-color Sets an element's background color - text will be highlighted in that color
opacity Can make an element transparent or semi-transparent where 1 is fully opaque, 0 is fully transparent and any decimal in between is semi-transparent
background-image Sets the background of an element with an image rather than a color
CSS Fonts/Typography
font-family Defines the typeface of an element, can list multiple as back-ups but first listed will be preferenced if available on the user's browser
font-size Controls the size of the text displayed
font-weight Defines how thick or thin text is displayed - number ranging from 100-900 (light-bold) with 400 being standard size
text-align Places the text in the left, right or center of its parent container
line-height Allow the vertical space between lines of text to be adjusted
word-spacing Changes how far apart individual words are
letter-spacing Changes how far apart individual letters are
The Box Model & Positioning
margin Amount of space outside an element's border, setting margin to "0 auto" horizontally centers an element inside of its parent content area, if it has a width
padding the space between the content area and the border. It can be set in pixels or percent
border surround the content area and padding of an element. The color, style, and thickness of a border can be set with CSS properties
overflow can be set to dispaly, hidden, or scorll and dictates how HTML will render content that overflows its parent element
visbility can hide or show elements
box-sizing Controls the box model used by the browser. Default is "content-box", setting it to "border-box" will not affect border thickess or padding
position Allows you to specify position of element on page - common postions are static, absolute, relative, fixed, sticky
display allows you to control how an element flows vertically and horizontally in a document - commonly used are block, inline and inline-block
z-index specifies how far back or how far forward an element appears on the page when it overlaps other elements
float can move elements as far left or as far right as possible on a web page
clear You can clear a floated element's left or right side or both using the clear property
CSS color type CSS color syntax
CSS Colors
Named Colors There are more than 140 colors that you can review Here on MDN
Hexidecimal or Hex Colors Six digit number system with 16 digits 0-9 and A-F (#23F41A), starts with #, specifies red, blue and green, if all three values are duplicate then it can be shortened to three digits
RGB Colors Use rgb() syntax with on value for red, blue and green, values range from 0-255 and look like this rgb(7,210, 50)
HSL colors HSL stands for hue, saturation and lightness, hues range from 0-360 and saturation and lightness are represented as a percentage
Opacity can be added to RGB and HSL colors by adding a 4th value "a" with is a percentage
